Lesson 1: The Basics of the Web and HTML

How the Web Works

The world wide web is a collection of HTML. The World wide web = collection of documents. HTML = hypertext markup language.

Users access the internet through their browser (examples: Safari, Chrome, Firefox etc...). The browser accesses a website by entering the URL which is the website (example: google.com) The users browser sends the HTTP request to the server which is storing the website as an HTML. The browser reads the HTML documents and displays it for the user.

Lesson 3: Adding CSS Style to HTML Structure

Avoiding Repetition

Avoiding repition is efficient for the programmer and also reduces the likelihood of errors within the code. Fixing errors or bugs is also made easier by reducing repitition.

Consistency is another practice to increase efficiency and reduce errors. When the coder knows certain topics of categories will have the same styling, then assign each lesson the same class (div and class).

CSS

CSS stands for Cascading Style Sheets and it controls the style of the web page. This is done by giving similar HTML elements the same class name and then specifying the style that should apply to that class.

CSS is used for styling and the "span style" can be used to specify the colors.
